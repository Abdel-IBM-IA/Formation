Le format de fichier XHTML Website

          Les fichiers enregistrés avec l'extension XHTML sont des fichiers de site web écrits dans un certain langage de programmation.
          Le contenu du fichier est écrit en XHTML qui est une transposition en XML 1.0 du HTML 4. 
          De nombreuses pages web sont désormais écrites en XHTML 1.0 ou XHTML 1.1, par opposition à la version HTML brute.

Détails techniques relatifs aux fichiers XHTML

          Les fichiers écrits à l'aide du langage de balisage XHTML utilisent des balises HTML standard. 
          Toutefois, la syntaxe plus rigoureuse du XML est reprise dans le XHTML, par opposition au code HTML pur. 
          En conséquence, les fichiers XHTML sont très bien structurés et peuvent facilement être analysés en utilisant le XML. 
          Depuis janvier 2000, le langage de programmation XHTML est devenu une recommandation du W3C
